## Description

"This is a desktop application that allows users to create and automatically sync notes based on work items from company hubs (such as work items from DevOps)"

## Download

"The latest version of this project is tag V1.0.0 in Realeaes in Github https://github.com/LanceBalstad/quick-notes/tags"

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/LanceBalstad/quick-notes.git

   ```

2. Install Requirements

## Requirements

- **Node.js** (v18+) and **npm** – [Download here](https://nodejs.org/)
- **Rust** – [Install from rustup.rs](https://rustup.rs/)

## Main Dependencies

**Frontend:**

- React 19, React Router DOM – UI framework
- Tauri – Desktop app framework
- Quill – Rich text editor
- Dexie – IndexedDB database

**Backend (Rust):**

- OAuth2, Reqwest – API and authentication
- Serde – JSON serialization
- Tokio – Async runtime

3. Install Dependencies
   Run `npm install` to install all required dependencies:

4. Running the Application
   To start the development server, run: 'npm run tauri dev'

   ```bash
   npm run tauri dev

   ```

   The application will be available as a desktop application.
